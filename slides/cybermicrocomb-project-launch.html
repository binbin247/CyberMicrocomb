An interactive browser-based LLE simulator for exploring microcomb dynamics, soliton states, Raman physics, and multi-field coupling.

Motivation Fast intuition before production-grade simulation
Why this matters

Microcomb physics is visual, nonlinear, and parameter-sensitive.

01

Setups slow down intuition

Local Python stacks, backend services, and lab-specific scripts are powerful but heavy for quick exploration.

02

Dynamics need live feedback

Detuning scans, dispersion changes, Raman terms, and mode coupling are easier to understand when plots respond immediately.

03

Models need context

Equations, normalization, physical pictures, demos, and references should sit next to the simulator itself.

Architecture Static page, local numerical engine
How it runs

After load, the browser becomes the lab bench.

React interface

Model selector, parameter controls, bilingual docs, export button, and live plots.

Web Worker

Solver runs away from the UI thread, keeping interaction responsive.

Pyodide + NumPy

Split-step LLE evolution, adaptive aliasing guard, multi-field snapshots.

Plots + JSON

Temporal field, comb spectrum, energy traces, waterfall history, and fields.

No Flask. No FastAPI. No server-side compute.
Model library Six normalized interactive models
Built-in physics

One interface, multiple routes into soliton microcombs.

Standard soliton

Single-field LLE for bright dissipative Kerr solitons in anomalous dispersion.

Dark pulse / platicon

Normal-dispersion LLE with local mode shift for platicon formation.

Stokes soliton

Primary-Stokes coupled LLE for Raman-assisted secondary soliton growth.

Turnkey soliton

Self-injection-locking model with a live phase diagram and locked detuning.

Multicolor soliton

Primary, signal, and idler fields for interband multicolor soliton dynamics.

Raman SSFS

Explicit Raman-response convolution for soliton self-frequency shift.

Standard soliton

Single-field normalized Lugiato-Lefever equation for bright dissipative Kerr solitons.

∂tψ = [-(1+iα) + i|ψ|2]ψ + i d2∂φ2ψ/2 + F

Suggested launch demo

Show a real physics transition in under two minutes.

01

Open the static GitHub Pages site

Pyodide loads once; then the numerical loop runs locally in the browser.

02

Select a model and scan detuning

Use Standard soliton, platicon, Stokes soliton, turnkey locking, multicolor, or Raman SSFS depending on the story.

03

Read the plots as the state evolves

Temporal field, comb spectrum, intracavity energy, and waterfall reveal stability, drift, and transitions.

04

Export complete data

Use JSON output for later analysis, reproducibility, figures, or teaching material.
